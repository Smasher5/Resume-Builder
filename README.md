# Resume Builder

A full-stack **Resume Builder Web Application** that allows users to create, edit, and manage professional resumes easily. The platform provides a clean UI where users can input their details, organize resume sections, and generate structured resumes ready for job applications.

---

## 🚀 Features

- User authentication (Register & Login)
- Create and manage resumes
- Add sections like:
  - Personal Information
  - Education
  - Experience
  - Skills
  - Projects
- Upload profile image
- Edit and delete resume entries
- Responsive UI
- REST API powered backend
